Task 1 Completed!

I have successfully completed Task 1, creating a simple, responsive landing page using HTML and CSS. The project focused on building a clean and functional website with a header, a hero section, and a footer. I used

Semantic HTML for a clear and accessible document structure and applied CSS Flexbox to manage the layout of the header and footer. To ensure the landing page is accessible on different devices, I implemented

media queries, which adjust the layout for smaller screens, such as collapsing the navigation menu. This task was an excellent opportunity to apply core web development concepts like the

CSS Box Model, responsive design, and the fundamental differences between padding and margin. I am excited to continue building on these skills in future tasks.
